## 2. 진단기간의 3가지 구분

대한정형외과학회 「사지 및 척추골관절의 상해진단 작성의 지침」에 따르면, 진단기간은 **3가지**로 구분됩니다:

| 구분 | 정의 | 비고 |
|------|------|------|
| **고정기간** | 골절부의 고정기간 (immobilization or surgical care) | 석고고정, 부목 등 |
| **치료종결기간** | 의사의 치료 및 관찰이 종료되는 기간 (observation and treatment ended) | 고정기간보다 김 |
| **재취업기간** | 재취업에 필요한 기간 (able to begin work) | **고정기간의 약 2배** |

> **핵심:** 상해진단서에 기록되는 진단기간은 **"고정기간"**을 기준으로 합니다.
> 치료종결기간이나 재취업기간을 진단기간으로 하면 통상의 진단기간보다 훨씬 길어지는 문제가 있습니다.

## 3. 진단주수 산정 기본 원칙

| 원칙 | 설명 |
|------|------|
| 단위 | **주(週)** 단위로 기재 |
| "전치 ○주"의 의미 | **일상생활에 지장이 있는 기간** (기능장애 기간) |
| 골절의 경우 | **석고고정기간(cast immobilization period)** = 진단기간 |
| 타박상(좌상) | 자연치유 되더라도 **01주**로 기재 |
| 찰과상 | **01주** |
| 물리치료 | 진단기간에 **불포함** |
| 복수 과목 진단 | **가장 중한 주수 하나만** 인정 (합산 불가) |

> **예시:** 신경외과 12주 + 정형외과 10주 → 최종 **12주** 인정

## 4. 연령별·손상정도별 보정 기준

### 연령별 보정률

대퇴골 간부골절 고정기간 14주, 쇄골 단순 골절 고정기간 5주 등이 연령 구분 없이 단일화되어 있는 문제를 보완하기 위한 보정 기준:

| 연령 구분 | 표준 진단기간 대비 비율 |
|---------|---------------------|
| **15세 미만 (소아)** | **80%** |
| **15세 이상 ~ 55세 미만** | **100%** (표준) |
| **55세 이상** | **120%** |

> 예: 대퇴골 간부골절 표준 14주 → 소아 11.2주(≒11주), 55세 이상 16.8주(≒17주)

### 골절 손상정도별 가산률

| 손상 유형 | 표준 진단기간 대비 비율 |
|---------|---------------------|
| 단순 골절 (표준) | **100%** |
| **전위** 골절 | **+10%** (110%) |
| **개방성** 골절 | **+20%** (120%) |
| **분쇄** 골절 | **+10%** (110%) |
| 개방성 + 전위 골절 | **130%** |
| 개방성 + 전위 + 분쇄 골절 | **140%** |

> 예: 상완골 간부 표준 9주 → 개방성+전위+분쇄 골절 시 9주 × 140% = 12.6주(≒13주)

## 7. 상해등급 영역별 세부지침

### 공통

- **병급:** 2~11급 중 2가지 이상 상해 중복 시, 가장 높은 등급부터 하위 3등급 사이의 상해가 중복된 경우에만 한 등급 상향
- 일반 외상 + 치과보철 중복 시: 각각의 등급별 금액 배상 (합산액이 1급 한도 초과 불가)
- 소아: 만 13세 미만

### 두부

- **두피 좌상, 열창** → 14급
- 만성 경막하 혈종 수술 → 6급 2호 적용
- 외상 후 급성 스트레스 장애: 단독 상병으로 외상 후 1개월 이내 발병 시만 적용

### 흉·복부

- **심장타박(6급)** 인정 요건: ① 심전도 이상 ② 심초음파 소견 ③ 심장효소치 증가 — 3가지 모두 충족

### 척추

- 완전 마비: 근력등급 3 이하 / 불완전 마비: 근력등급 4
- 척추관 협착증·추간판 탈출증이 외상으로 발생/악화 → **9급**
- 척주 손상으로 신경근증·감각이상 호소 → **9급**
- 마미증후군 → 척수손상으로 봄

### 상·하지 공통

- **보존적 치료 골절:** 해당 등급에서 **2급 하향**
- **도수정복 + 경피적 핀고정:** 해당 등급에서 **1급 하향**
- **개방성 골절/탈구** (거스틸로 2형 이상, 개방창 1cm 이상): **1등급 상향**
- 골절 + 주요 말초신경 손상 동반: **1등급 상향**
- **소아:** 성인 동일 부위 골절보다 **1급 낮게** (단, 성장판 손상 동반 시 성인과 동일)
- 근·건 부분파열 보존적 치료 → 단순 염좌 (12급)
- 보존적 치료 사지 주요관절 골절 및 탈구: 해당관절보다 **3등급 하향**
- 수술 시행 사지 주요 관절 탈구: 보존적 탈구보다 **2등급 상향**

### 상지

- 회전근개 파열: 개수에 따른 차등 없음
- 견봉쇄골인대 + 오구쇄골인대 완전 파열 수술 → 7급
- 부분 파열 보존적 치료 → 9급
- 단순 염좌 → 12급

### 하지

- 천골/미골 골절 → 골반골 골절로 봄
- 경골 후과 단독 골절 → 족관절 내과/외과 골절로 봄
- 슬관절 십자인대 파열: 완전파열(또는 이에 준하는 파열)로 인대 복원수술 시행 시 적용
